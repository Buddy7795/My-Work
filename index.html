HTML Colors

HTML colors are used to set the color of text, background, and other elements on a web page.

HTML colors can be specified using color names, hex codes, or RGB values.

  • Color names
    • Ex: red, green, blue, yellow, orange, purple, pink, brown, black, white

    There are 3 HTML colors

    1. Hex codes
      2. Ex:#FF0000 (red), #00FF00 (green),#0000FF (blue)
    2. RGB values
      3. Ex: rgb(255, 0, 0) (red),rgb(0, 255, 0) (green), rgb(0, 0, 255) (blue)
    3. HSL colors

      4. The HSL(Hue, Saturation, Lightness) function accepts 3 values

      • A number from 0 to 360 for Hue
      • A percentage from 0(grey) to 100(standard) for Saturation
      • A percentage from 0(black) to 100(white) and 50(normal) for Lightness
      Ex: hsl(0, 100%, 50%) (red), hsl(120, 100%, 50%) (green), hsl(240, 100%, 50%) (blue)
